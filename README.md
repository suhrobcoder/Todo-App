# Todo App
This app is a simple to-do list app that helps you get things done.

## Libraries used in this project
* Coroutines
* Room
* LiveData
* ViewModel
* Navigation Component
* Dagger-Hilt
* Glide
* Firebase (Auth, Firestore, Storage)
